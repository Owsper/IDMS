User Guide

Dashboard

Core Workflows

  • Register, verify email, sign in, and maintain your member profile from Profile.
  • Upload approved documents from Documents and search by title or category.
  • Open Voting to cast one secure vote per active event when your membership is eligible.
  • Open Meetings to view upcoming meeting cards, agendas, locations, and minutes.
  • Use Bugs to report defects with severity, reproduction steps, expected behavior, and actual behavior.

Admin Workflows

  • Create voting events with candidates, date validation, and eligibility rules.
  • Import WhatsApp .txt exports, then review daily activity, top participants, peak hours, and message type ratios.
  • Schedule meetings, record attendance, save minutes, and export attendance CSV reports.
  • Record income and expenses, create budgets, monitor threshold alerts, and export financial CSV reports.
  • Use Activity Summary to review meetings, votes, documents, transactions, and the cross-module timeline.

Testing Plan

System testing covers happy paths, validation failures, access restrictions, duplicate prevention, malformed imports, empty reports, export generation, and regression checks after bug fixes. Each module has focused unit tests for creation, verification, analysis, report generation, and status workflows.
