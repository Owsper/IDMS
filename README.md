# IDMS

A grouped project for Frontend and Agile Dev.

## Project Layout

- `src/` - Flask app, database helpers, and Python unittest suites.
- `frontend/pages/` - Jinja HTML templates.
- `documentation/` - user, technical, screenshot, and system testing docs.
- `secure_uploads/` - local runtime upload storage.

## Run Tests

```powershell
python -m unittest discover -s src
```
