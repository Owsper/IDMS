## Capture Notes

- Use an administrator session for admin-only screens.
- Use a verified member session for member-only screens.
- Capture at 1440px desktop width and repeat key workflows at a mobile width when layout changes materially.
- Redact passwords, tokens, real email addresses, and private uploaded document names.
- Re-capture screenshots after major UI changes.
